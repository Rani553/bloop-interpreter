# BLOOP Interpreter (Java)

A Java-based academic project focused on building an interpreter for a custom programming language called **BLOOP**.
The project is designed to strengthen understanding of how programming languages work internally through the **tokenization, parsing, and execution pipeline**.

---

## 🎯 Project Objective

* Build a working interpreter from scratch
* Understand the **Tokenizer → Parser → Execution** workflow
* Practice **object-oriented design, AST handling, and execution flow**
* Learn collaborative development using **GitHub workflow and pull requests**
